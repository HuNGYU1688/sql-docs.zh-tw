# <a name="issdataclassification"></a>ISSDataClassification
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW](../../../includes/applies-to-version/sql-asdb-asa.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  **ISSDataClassification** 介面能提供擷取自 SQL Server 接收之作用中資料列集的敏感性分類資料的功能。
  

## <a name="methods"></a>方法

|方法|說明|  
|------------|-----------------|  
|[ISSDataClassification::GetSensitivityClassification](../../oledb/ole-db-interfaces/issdataclassification-getsensitivityclassification-ole-db.md)|傳回指向 SENSITIVITYCLASSIFICATION 結構、包含敏感性分類資訊的指標。|
